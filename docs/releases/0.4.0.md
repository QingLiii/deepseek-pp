## DeepSeek++ 0.4.0

0.4.0 在 0.3.0 的 Chrome / Edge / Firefox 多浏览器基线上，补齐本机 Shell / Office 文档工具、Agent 式持续执行、自动化任务触发、输出速度显示，并修复一批长任务稳定性问题。

### 主要变化

- 内置 OfficeCLI 官方 Skill 与 PPT 样式库，覆盖 DOCX、XLSX、PPTX、Pitch Deck、Academic Paper、Financial Model、Dashboard 和 Morph PPT 等文档场景。
- 新增 Shell MCP Native Messaging host 与安装脚本，可创建 Shell 预设，并通过 `shell_exec` / `shell_status` 调用本机命令工具。
- Shell 安装流程会优先检查命令版 OfficeCLI，并把用户本机工具目录放在项目依赖目录之前，避免误用 hosted 生成路径。
- 新增 Agent 式持续执行：MCP 工具结果支持回传到同一会话继续生成，让 DeepSeek 像 Claude Code / Codex 一样根据执行结果持续决定下一步。
- 连续执行过程按 Step 折叠展示，并支持执行中停止和刷新后恢复。
- 工具执行过程和结果可以在页面刷新后恢复，减少长任务状态丢失。
- 回复生成时新增实时 `tok/s` 速度显示。
- 自动化任务继续保留「自动化」定位：它负责立即运行和 cron/RRULE 定时触发；触发后的任务可复用预设、记忆、MCP 工具和 Agent 式续跑链路。
- 修复流式取消、计时器清理、工具解析计数、多片段偏移、工具记录恢复和 mock catalog 差异等问题。
- README 新增 0.4.0 变更回顾、OfficeCLI/Shell MCP 说明、工具续跑与速度显示示意图，并恢复自动化任务命名。
